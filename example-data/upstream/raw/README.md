# Raw FASTQ files (upstream input)

Paired-end 16S reads for 6 samples (3 Control + 3 Drought), subsampled to 10,000
read pairs each. `_r1` = forward, `_r2` = reverse.

See **`example-data/README.md`** (two levels up) for full instructions on running
these through the EzMAP2 upstream pipeline and on to downstream analysis.
