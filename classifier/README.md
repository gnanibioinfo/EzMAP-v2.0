# Classifier Reference Data

The full taxonomy classifiers are too large to host in the GitHub repo.

Download the complete EzMAP v2.0 package — including this folder's contents — from the Releases page:

https://github.com/gnanibioinfo/EzMAP-v2.0/releases/latest

Required files in this folder:
- `silva-138-99-nb-classifier.qza` (~208 MB) — full-length SILVA 138 Naive Bayes classifier
- `silva-16S-V4-nb-classifier.qza` (~154 MB) — V4-region SILVA 138 classifier
- `unite-ver7-99-classifier.qza` (~46 MB) — UNITE ITS classifier (fungi)
